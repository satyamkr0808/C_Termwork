# README: Term Work PCS 252

[cite_start]This document outlines the requirements and problem sets for the **PCS 252 Term Work**[cite: 1]. [cite_start]It serves as a guide for students to complete their 
C programming assignments in accordance with specific academic standards[cite: 1].

---

## Submission Guidelines

To ensure full credit, all solutions must follow these physical and structural requirements:

### Physical Presentation
* [cite_start]**Stationery**: Use thick, one-sided ruled sheets for writing and DMP (Dot Matrix Printer) sheets for code printouts[cite: 3].
* [cite_start]**Flowchart Margins**: Flowchart sheets must have a **2cm** margin on the right and **1cm** margins on the remaining three sides[cite: 6].
* [cite_start]**Code Headers**: Every program must include the student's **Name**, **University Roll No**, and **Section** within the comment section alongside the problem statement[cite: 7].

### Solution Components
The required documentation varies depending on the question number:

| Questions | Required Components |
| :--- | :--- |
| **1 – 13** | [cite_start]Problem statement, Assumptions, Algorithms, Flowchart (on the back of the ruled sheet), Program printout, and all possible outputs[cite: 4]. |
| **14 – 16** | [cite_start]Problem statement, Assumptions, Algorithms, Program printout, and all possible outputs (Flowcharts are not required for these)[cite: 5]. |

## Problem Set Overview

### Basic Logic and Sequences
* **1. Quadratic Roots**: Calculate roots of a quadratic equation. [cite_start]For example, coefficients 4, 1, and 0 should yield roots 0.00 and -0.25[cite: 8, 9, 10].
* **2. [cite_start]Character Identification**: Determine if an input character is a number, alphabet, or special symbol[cite: 11].
* **3. [cite_start]Fibonacci Series**: Print the series up to $N$ terms using **recursion**[cite: 12].
* **4. [cite_start]Sum of Series**: Calculate the sum of the series $1^{2}-2^{2}+3^{2}-4^{2}$ up to $n$ terms[cite: 13, 14, 15].
* **5. Pattern Generation**: 
    * [cite_start]Alphabet triangle (A, BC, DEF...)[cite: 16, 17, 18, 19, 20, 21].
    * [cite_start]Binary triangle (1, 01, 101...)[cite: 22, 23, 24, 25, 26].

### Arrays, Strings, and Matrices
* **6. [cite_start]Array Reversal**: Insert $n$ elements and replace each with its reverse[cite: 27].
* **7. [cite_start]Anagram Checker**: Verify if two strings contain the same characters with the same frequency[cite: 28, 29].
* **8. [cite_start]Symmetric Matrix**: Check if a square matrix is equal to its transpose[cite: 30, 31].
* **9. [cite_start]Saddle Point**: Find an element that is the minimum in its row and maximum in its column[cite: 32, 33, 34].

### Pointers and Memory Management
* **10. [cite_start]Email Validation**: Use **pointers** to perform a case-insensitive comparison between a user-input email and a statically initialized ID[cite: 35, 36, 37].
* **11. [cite_start]Dynamic Array**: Use **Dynamic Memory Allocation** to input $n$ integers and insert a new element at a user-specified position[cite: 38, 39].

### Structures
* **12. [cite_start]Book Records**: Sort a list of books (ID, Title, Author, Price) in **descending order** of price[cite: 40, 41].
* **13. Employee Management**: 
    * [cite_start]Display employees with a salary above a specific threshold[cite: 42, 43].
    * [cite_start]Search for specific employee details using an ID[cite: 44].

### File Handling
* **14. [cite_start]Prime Sorter**: Read integers from `numbers.txt` and separate them into `prime.txt` and `non_prime.txt`[cite: 45, 46].
* **15. [cite_start]Text Processor**: Read `Input.txt`, remove all digits, and write the count of vowels and consonants to `Output.txt`[cite: 47].
* **16. [cite_start]Product Inventory**: Store product details in `product.txt` and filter those with a total cost exceeding 10,000 into `expensive.txt`[cite: 48, 49].
